Data Hubs
=========

Data-hubs combine functions for easy loading and processing of SimFin data.
See `Tutorial 05`_ for detailed examples.

For convenience, the data-hubs are also available directly in the `simfin`
namespace, so instead of using `simfin.hubs.StockHub` you can simply use
`simfin.StockHub`
