Signals
=======

These functions are used for calculating signals such as P/E and P/Sales ratios,
Sales Growth, Return on Equity, etc. See `Tutorial 04`_ for detailed examples,
but note that it is often much easier to use :doc:`hubs` for calculating these
signals.
